Java 迷宫求解：
1. 主要有三个子问题需要求解：
        ①  迷宫生成算法： 通过在网上或图书资源查阅并实现；
        ②  迷宫求解算法： 通过回溯算法实现；
        ③  迷宫及解的显示：采用动画显示。
        
2. 程序结构采用观察者模式。 
         ①    GUI 界面负责显示数据和事件激发；一个 MODEL 对象对按钮事件进行注册监听；
         ②    迷宫对象 Maze 作为被观察者对象， 迷宫面板对象作为观察者； 
         ③    操作系统是隐藏的控制器； 当按钮按下时，通过系统底层事件机制触发 MODEL对象改变，并通知面板对象更新；
         ④    MODEL 对象从GUI界面接收迷宫数据并进行迷宫求解，将求解数据分别推送给面板进行更新；
         ⑤    一个 GUI 界面可与 一个或多个 MODEL 相关联，在 MODEL 对象中引用，以便从中获取要处理的用户输入。
